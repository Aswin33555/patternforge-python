# 🧠 Number Pattern Generator

A modern Python CLI application that generates beautiful number patterns using loops and modular programming concepts.

---

## 🚀 Project Overview

This project was developed as part of a Software Development Internship task to demonstrate:

* Looping concepts
* Nested loop structures
* Conditional statements
* Function-based programming
* User interaction in CLI applications

The application allows users to generate multiple number patterns dynamically through a clean menu-driven interface.

---

# ✨ Features

✅ Pyramid Pattern Generator
✅ Reverse Number Pattern

✅ Beginner-Friendly Architecture
